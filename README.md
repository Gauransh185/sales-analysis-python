
# Data Analyst Project: Retail Performance Analysis – Sales and Profitability Insights

## Project Description  

This project analyzes retail transaction data to uncover insights into sales performance, profitability, and customer behavior.

The goal is to transform raw transactional data into meaningful insights that can help businesses make data-driven decisions. The analysis focuses on identifying high-performing products, understanding the impact of discounts on profit, and evaluating regional performance.

## Dataset 

The dataset contains retail transaction records including features such as `sales`, `profit`, `discount`, `category`, and `region`.

## Tools & Technologies Used  

- **Python** (Data Analysis and Logic Building)  
- **Pandas** (Data Cleaning, Transformation, and Analysis)  
- **Matplotlib** (Data Visualization and Trend Analysis)

## Key Insights  

- **Technology Category:** Generated the highest overall sales among all categories  
- **High Discounts:** Orders with higher discounts showed a noticeable decline in profit margins  
- **Regional Trends:** Certain regions achieved high sales volume but relatively low profitability  
- **Outliers identified:** Specific cases in sales and profit were identified, which may influence overall aggregated results
- **Product Performance:** A subset of products consistently performed well across multiple regions

## Conclusion  

The analysis highlights important business trends and provides insights that can help improve pricing strategies, product focus, and regional performance. It demonstrates the practical application of data analysis in solving real-world business problems and improving decision-making.
